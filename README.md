# Digital Store
## Tecnologias
Projeto criado com:
* Tailwind
* Vite.js
* MySQL
* Sequelize
## Inicializar
Para executar esse projeto, execute:

```
$ cd ./frontend
$ npm install
$ npm run dev
```

Para iniciar o servidor, configure o arquivo .env.sample e execute:

```
$ cd ./backend
$ npm install
$ npm run sync
$ npm run server
```
